# Visual Assets

## Perma-Temp Punks Cover

`perma-temp-punks-cover.png` is an exploratory cover concept generated with OpenAI image generation for Bitpixi's novel.

The image depicts Phosphor and Chapel Seven in a Victorian Goldfields rig convoy, with Eris and Red Flag nearby, the Undermesh beneath them, and the fixed office grid opening above the landscape.

The cover is concept art rather than final publication artwork. Typography, trim, credits, and print colour would need a dedicated production pass before commercial use.
