# Officecore 2064 Game-Offshoot

This folder is a playable story-game offshoot for `officecore-2064`.

It turns the CyberCamper Village Swarm seed into a collaborative table kit: people, agents, and ridiculous Aussie campervans trying to keep each other alive, online, housed, credited, flirted with, and only mildly audited in 2064.

The game borrows broad tabletop habits like character sheets, lifepath questions, specialty roles, vehicle identity, shared resources, net/data-centre jobs, and encounter sheets. It does not use another game's setting, factions, prose, combat focus, or mechanics.

## What Is Included

- [docs/story-kit.md](docs/story-kit.md): the core tone, table flow, rules, resources, and facilitator guidance.
- [docs/character-creation.md](docs/character-creation.md): campervan/character creation, stats, specialties, bonds, secrets, and faults.
- [docs/encounters/001-bushfire-edge-cache.md](docs/encounters/001-bushfire-edge-cache.md): the starter encounter about a bushfire, an edge data centre, and a crew that cannot stay parked.
- [data/officecore-kit.schema.json](data/officecore-kit.schema.json): a future-friendly schema for characters, encounters, and crew state.

## Default Tone

Cyberpunk-adjacent, Australian, funny, flirty, collaborative, sponsor-safe, and officecore.

Keep the focus on:

- mobile offices and improvised civic life
- data centres, mesh networks, weird paperwork, and care ledgers
- campervans with personality and specialty areas
- banter, chemistry, and consent-forward social play
- practical crises solved by odd people being competent together

Keep out:

- heavy-guns emphasis
- military power fantasy
- copied canon from other games
- misery-only dystopia
- crypto-bro triumphalism

## Quick Start

1. Each player makes one person plus one campervan using the character guide.
2. The table chooses a shared rig town or convoy name.
3. Set shared resources to:
   - `Charge 3`
   - `Trust 3`
   - `Smoke 1`
   - `Heat 1`
   - `Signal 3`
4. Run the starter encounter.
5. End with one new module, one new relationship complication, and one unresolved office problem.

## Core Roll

When the outcome matters, roll `2d6 + one stat`.

- `10+`: clean success.
- `7-9`: success with a cost, delay, attention, awkwardness, or resource drain.
- `6 or less`: a complication lands, but the crew learns something useful.

Do not roll for basic competence. Roll when pressure, risk, comedy, attraction, public accountability, or unreliable infrastructure would make the result interesting.
