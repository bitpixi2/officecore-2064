# Officecore 2064 Novel Test - Bitpixi

This folder is Bitpixi's prose test area for `officecore-2064`.

Use it for rough novel material, scene fragments, voice tests, chapter sketches, character moments, and experiments that might later become part of the wider Officecore 2064 fiction project.

## Working Boundary

This is not final canon by default. It is a labelled test space so readers and contributors know the material is currently Bitpixi-led and exploratory.

## Useful Starting Points

Future files can include:

- `notes.md` for loose story direction
- `chapter-001.md` for a first prose attempt
- `characters.md` for recurring figures and relationship notes
- `locations.md` for rig towns, service stations, data centres, motels, depots, and other officecore spaces

## Contributor Boundary

This folder follows the repo's existing content-license intent. Contributions remain contributor-owned unless separate written terms say otherwise. If material from this folder becomes part of commercial publication, credit and terms should be handled explicitly first.
