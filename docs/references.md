# References And Inspiration

This project is original fiction. It can be inspired by tabletop roleplaying collaboration, cyberpunk media, office culture, Australian infrastructure, housing precarity, and retro technology without copying protected text, tables, art, settings, or mechanics.

## Cyberpunk 2020 / Cyberpunk RED

Kasey mentioned *Cyberpunk 2020* as inspiration for the homebrew feel: character/crew design, customised gear, social roles, and collaborative play.

Do not import pirated PDFs or scanned rulebooks into this repo.

Legitimate places to look:

- R. Talsorian Games Cyberpunk page: https://rtalsoriangames.com/cyberpunk/
- R. Talsorian downloadable content: https://rtalsoriangames.com/downloadable-content/
- Cyberpunk 2020 on DriveThruRPG: https://www.drivethrurpg.com/en/product/50354/cyberpunk-2-0-2-0-the-second-edition-version-2-01
- Cyberpunk RED Easy Mode on DriveThruRPG: https://www.drivethrurpg.com/en/product/409912/cyberpunk-red-easy-mode

If you own legal copies, use them privately for inspiration. When contributing here, write original material in the `officecore-2064` setting.
